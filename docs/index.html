Input

Input

Input is a superclass for Linus input methods.
All input methods should derive from this class.
If any subclass implements diferent methods, those should be private.

Public methods

.input()
returns: String
Implement this method in the subclasses to define how the input is obtained.

MicrophoneInput

MicrophoneInput is a voice based input method.

Constructor

MicrophoneInput([timeout, phraseTimeLimit])
(Integer) timeout: Maximum amount of time the microphone has to wait for an input.
(Integer) phraseTimeLimit: Maximum time lenght of a phrase.

Public methods

.input()
returns: String
Gets input from the microphone and returns what was said.

RawInput

Constructor

RawInput([arg])
(String) arg: Message that shows off before writing something.

Public methods

.input()
returns: String

Gets input from the standard input and returns what was written.

Adding new Input methods

To add new input methods you need to first create a new class that extends the Input class and implement the .input() method. Next place this class in the folder src/input and import it as needed.

Output

Output

Output is a superclass for Linus output methods.
All output methods should derive from this class.
If any subclass implements diferent methods, those should be private.

Public methods

.sendOutput(arg)
returns: Void
(String) arg: Message to output to the user.

Implement this method in the subclasses to define how Linus sends information to the user.

Adding new Output methods

To add new output methods you need to first create a new class that extends the Output class and implement the .sendOutput() method. Next place this class in the folder src/output and import it as needed.

Intents

label=None, file=None, classifier= None

Intent

An Intent represents the purpose of the user towards Linus. This is an "abstract" class, therefore other subclasses should implement diferent intents.

Constructor

Intent(label, file, classifier)
(String) label: Maximum amount of time the microphone has to wait for an input.
(String) file: Name of the train file that contains all training data for the intent.
(Classifier) classifier: The instance of the classifier responsible for the classification of all intents.

Public methods

.execute(linus, [*args])
returns: String or None
(Linus) linus: The instance of linus.
(Anything) *args: Any other needed arguments.

Implement this method in the subclasses to define what the intent does.

.respond(inputReceived)
returns: String
(String) inputReceived: The input that the user gave to Linus.

Gets input from the microphone and returns what was said.
